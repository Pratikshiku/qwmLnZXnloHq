### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、志愿者、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 宠物信息
5. 宠物领养
6. 论坛
7. 公告信息
8. 个中心
9. 宠物收藏

**志愿者功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 宠物信息管理
6. 宠物寄养管理
7. 宠物领养管理
8. 论坛管理
9. 公告信息管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 宠物信息管理、宠物留言管理、宠物收藏管理
5. 宠物寄养管理
6. 宠物领养管理
7. 宠物领养审核管理
8. 基础数据管理（宠物类型管理、公告类型管理）
9. 论坛管理
10. 公告信息管理
11. 用户管理
12. 志愿者管理
13. 轮播图信息
